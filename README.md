## 🎮 Interactive Features & Mechanics

This portfolio is not just a static webpage; it is a full **3D physics sandbox** and game-like experience built directly inside the browser. Key highlights include:

*   **Gaming Style Welcome Screen:** Visitors are greeted with a retro Pacman loading animation and a dynamic, hover-reactive **Batman UI Enter Button** using CSS mask rendering.
*   **3D Physics Background (Three.js & Cannon.js):** Real-time physical bodies (spheres and boxes) fall, collide, and interact with the cursor in a fully simulated 3D space.
*   **Interactive Torchlight Toggle:** A physical torch toggle element allows the user to cast a dynamic virtual spotlight across the webpage to reveal hidden details in a dark-themed aesthetic.
*   **GSAP ScrollTrigger Timelines:** Ultra-smooth, scroll-bound transitions that animate web sections into view as the user scrolls, maintaining cinematic pacing.
*   **Dynamic Custom Cursor:** A sleek particle-following pointer that replaces the default browser cursor and interacts with buttons, cards, and links.

---

## 🛠️ Tech Stack & Libraries

*   **3D Graphics:** [Three.js](https://threejs.org/) for rendering the WebGL canvas, lights, cameras, and geometries.
*   **Physics Engine:** [Cannon.js](https://github.com/schteppe/cannon.js) for calculating gravity, mass, friction, and rigid body collisions.
*   **Animations:** [GSAP (GreenSock Animation Platform)](https://greensock.com/) & [ScrollTrigger](https://greensock.com/scrolltrigger/) for advanced scroll-driven timeline mechanics.
*   **Design Frameworks:** Custom raw CSS design system containing fluid grid structures, glassmorphism containers, and interactive UIverse components.

## 🚀 How to Run Locally

Since the portfolio is built on clean, native web technologies (HTML, CSS, JS), you can run it instantly without complex server setups:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/karnavignesh/portfolio.git
   cd portfolio
   ```
2. **Open the page:**
   * Double-click `index.html` to open it in your browser.
   * *Alternative:* Right-click `index.html` in VS Code and select **Open with Live Server** to enable hot reloading.

## 🌐 Deployment

To host this repository live for free on **GitHub Pages**:
1. Push this repository to your GitHub account.
2. Navigate to your repository on GitHub and go to **Settings** > **Pages**.
3. Under **Build and deployment** > **Source**, choose **Deploy from a branch**.
4. Select the **main** branch and folder `/ (root)`, then click **Save**.
5. Your live URL will be ready at: `https://karnavignesh.github.io/portfolio/`
